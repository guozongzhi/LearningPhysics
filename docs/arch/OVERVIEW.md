# 🌐 LearningPhysics 项目概览 (Project Overview)

## 📖 项目简介
LearningPhysics 是一个专为高中物理设计的 AI 自适应学习平台。它结合了知识图谱与生成式 AI (GenAI) 技术，为学生提供精准的错题归因诊断和个性化的学习路径。

---

## 🏗️ 系统架构

系统采用模块化单体架构，核心组件包括：

### 1. 前端 (Next.js 14+)
- **技术栈**: TypeScript, Tailwind CSS, Shadcn UI, KaTeX (公式渲染)。
- **功能**: 用户交互、实时答题、管理员仪表盘、知识雷达展示。

### 2. 后端 (FastAPI)
- **技术栈**: Python 3.11+, Async Native, Pydantic, SQLModel。
- **核心模块**:
  - `auth`: JWT 身份验证与权限管理。
  - `quiz`: 基于知识图谱的智能组卷与评分。
  - `admin`: 学生与题库管理接口。
  - `logs`: 统一的日志上报与追踪系统。

### 3. 数据层 (PostgreSQL + pgvector)
- **关系数据**: 存储用户、题目、测验记录及知识点关联。
- **向量检索**: 利用 `pgvector` 实现题目的语义搜索与推荐。

---

## 🛠️ 项目组织与管理

为了保持代码的可维护性，项目进行了深度的结构优化：

- **统一入口**: 根目录下的 `./manage.sh` 处理所有启动、停机、诊断及初始化操作。
- **资源外置**: 
  - `data/`: 存放题库 JSON 等非代码资源。
  - `config/`: 集中管理环境变量。
  - `docs/`: 分门别类存储设计、架构与调试文档。
  - `logs/`: 统一存储全链路运行日志。
- **环境隔离**: 使用项目根目录下的独立 `venv/` 虚拟环境。
